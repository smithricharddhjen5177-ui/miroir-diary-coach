# Journal Directory

Your diary entries are stored here as Markdown files (`YYYY-MM-DD.md`).

## Setup

Run the seed script to create demo entries:
```bash
npx tsx scripts/seed.ts
```

## Data Format

Each file uses YAML frontmatter for structured data and Markdown body for free writing.
The format is compatible with Obsidian, VS Code, or any text editor.

## Privacy

These files are local to your machine. They are gitignored by default.
You own your data — always.
